# Travel-Agency-Website
Technologies Used
HTML5 - Modern semantic HTML structure
CSS3 - Advanced styling with:
CSS Grid - For layout structure
Flexbox - For flexible component alignment
CSS Custom Properties (Variables) - For consistent theming
Media Queries - For responsive design
Google Fonts - Inter and Roboto font families
Pure HTML/CSS - No JavaScript frameworks or libraries
How to Run/View
This is a static website, so it's very simple to run:

Clone the repository:

bash
git clone https://github.com/Ondrej108/Travel-Agency-Website.git
Open locally:

Simply open the index.html file in any web browser
Or use a local server (Live Server in VS Code, Python's http.server, etc.)
View online:

You can enable GitHub Pages in the repository settings to host it for free
