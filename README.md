## üöÄ How to Run

This is a static website, making it simple to run locally or deploy online.

### Option 1: Open Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ondrej108/Travel-Agency-Website.git
   ```

2. **Navigate to the project folder**:
   ```bash
   cd Travel-Agency-Website
   ```

3. **Open in browser**:
   - Simply double-click `index.html`, or
   - Right-click `index.html` ‚Üí Open with ‚Üí Your browser

### Option 2: Use a Local Server (Recommended)

- **VS Code Live Server**: Right-click `index.html` ‚Üí "Open with Live Server"
- **Python**:
  ```bash
  python -m http.server 8000
  ```
  Then visit: `http://localhost:8000`

### Option 3: Deploy Online

- **GitHub Pages**: Enable in repository Settings ‚Üí Pages ‚Üí Source: main branch
- **Netlify**: Drag and drop the project folder
- **Vercel**: Import the GitHub repository
